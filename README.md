A simple set of JSON wrappers

### Maven Dependency
Add to your `pom.xml`
```xml
<dependency>
	<groupId>me.shib.java.lib</groupId>
	<artifactId>jsonutils</artifactId>
	<version>0.0.1</version>
</dependency>
```
